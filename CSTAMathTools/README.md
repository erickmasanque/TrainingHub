## 🔵🔴 Chip Whiteboard

An interactive whiteboard for demonstrating **addition and subtraction of integers** using colored chips.

**Features:**
- Blue chips (+1) and Red chips (−1) with drag-and-drop
- 🗑️ Trash bin — drag chips to delete
- ✏️ Drawable rectangles to mark zero pairs, answers, and groups (5 styles)
- ⚡ Auto zero-pair detection and animated removal
- Expression parser — type `(+3)+(-2)` and auto-place chips
- Running total display with color coding
- Undo history (Ctrl+Z)
- Works offline — no internet needed after first load

**Curriculum:** M7NS-Ic-d-1 — Performs fundamental operations on integers

## 📊 Lesson Slides

Animated HTML presentation for the Addition & Subtraction of Integers lesson (13 slides).

**Covers:** Objectives, Integer War card game, chip model introduction, zero pairs, addition/subtraction examples, group discovery activity, rules summary, and quiz.
